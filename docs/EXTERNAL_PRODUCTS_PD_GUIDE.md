# 외부 상품 - PD 활용 가이드

> PD 및 편성팀을 위한 가이드

---

## 개요

방송 편성 추천 시스템에 **네이버 인기 상품 + 타사 편성 정보**가 통합 제공됩니다.
외부 트렌드와 경쟁사 전략을 참고하여 더 나은 편성 결정을 내릴 수 있습니다.

---

## 제공 정보

### 네이버 인기 상품
- 네이버 쇼핑에서 가장 많이 구매하는 상품
- 매일 새벽 2시 자동 업데이트
- 현재 시장 트렌드 파악용

### 타사 편성
- 해당 방송 시간 ±1-2시간 내 타사 편성
- 경쟁사 전략 분석용
- 중복 편성 회피 또는 경쟁 전략 수립

---

## 활용 방법

### 1. 트렌드 파악

**네이버 인기 상품 활용:**
- 현재 시장에서 인기 있는 상품 카테고리 확인
- 유사 카테고리 자사 상품 편성 시 참고
- 계절/시기별 트렌드 변화 파악

**예시:**
> "가습기가 네이버 인기 상품에 올라왔네요. 겨울철 건조한 날씨 때문인 것 같습니다. 우리도 가습기 관련 상품을 편성하면 좋을 것 같아요."

### 2. 경쟁사 분석

**타사 편성 활용:**
- 같은 시간대 경쟁사가 어떤 상품을 편성하는지 확인
- 중복 편성 회피 또는 정면 경쟁 전략 수립
- 시간대별 경쟁 강도 파악

**예시:**
> "GS홈쇼핑이 22시에 건강식품을 편성했네요. 우리는 다른 카테고리로 가거나, 더 경쟁력 있는 건강식품을 편성해볼까요?"

### 3. 카테고리 분석

**확인 포인트:**
- 네이버 인기 상품에 많이 등장하는 카테고리
- 타사가 집중 편성하는 카테고리
- 경쟁이 적은 틈새 카테고리

## 체크리스트

편성 결정 전 확인사항:

- [ ] 타사 편성과 카테고리 중복 여부 확인
- [ ] 네이버 인기 상품과 유사한 자사 상품 있는지 확인
- [ ] 시간대별 경쟁 강도 파악
- [ ] 계절/시기에 맞는 상품인지 확인
