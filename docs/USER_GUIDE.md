# 📱 홈쇼핑 매출 예측 시스템 사용 가이드

> **대상**: 기획자, 방송 편성 담당자, 비개발자

---

## 🎯 이 시스템은 무엇인가요?

홈쇼핑 방송 편성을 도와주는 **AI 매출 예측 시스템**입니다.

### 할 수 있는 일
- ✅ 특정 상품을 특정 시간에 방송하면 **얼마나 팔릴지 예측**
- ✅ 여러 시간대를 비교해서 **가장 많이 팔릴 시간대 찾기**
- ✅ 신상품도 예측 가능 (과거 방송 이력 없어도 OK)
- ✅ 하루 전체 편성표의 예상 매출 확인

### 예측 정확도
- 평균적으로 실제 매출의 **±300만원** 오차
- 방송 횟수가 많을수록 더 정확
- 신상품은 카테고리 평균으로 예측 (정확도 낮음)

## 📊 주요 기능 설명

### 1️⃣ 단일 상품 매출 예측

**언제 사용하나요?**
- 특정 상품을 언제 방송할지 결정할 때
- 여러 시간대 중 최적 시간을 찾을 때

**입력 정보:**
- 상품 코드 (예: 15750903)
- 방송 날짜 (예: 2026년 2월 25일)
- 방송 시간 (예: 오후 2시)

**결과:**
- 예상 매출액 (예: 2,331만원)
- 신뢰도 (예: 85%)
- 상품 정보 (카테고리, 시간대 등)

**실제 예시:**
```
상품: 세일 토비콤 루테인 지아잔틴 12박스
날짜: 2026년 2월 25일
시간: 오후 2시

→ 예상 매출: 2,331만원 (신뢰도 85%)
```

### 2️⃣ 시간대별 비교

**언제 사용하나요?**
- 같은 상품을 언제 방송하면 가장 많이 팔릴지 알고 싶을 때

**비교 예시:**
```
상품: 세일 토비콤 루테인 지아잔틴 12박스
날짜: 2026년 2월 25일

오전 9시  → 예상 매출: 1,461만원
오후 2시  → 예상 매출: 1,026만원
저녁 6시  → 예상 매출: 1,373만원
저녁 9시  → 예상 매출: 1,736만원 ⭐ 최고!

결론: 저녁 9시에 방송하면 가장 많이 팔림
```

### 3️⃣ 하루 전체 편성표 예측

**언제 사용하나요?**
- 특정 날짜의 전체 편성표 매출을 미리 확인하고 싶을 때
- 하루 총 매출 목표 달성 가능 여부 확인

**결과:**
- 각 방송별 예상 매출
- 하루 전체 예상 총 매출

## 💡 실무 활용 시나리오

### 시나리오 1: 신상품 편성 시간 결정

**상황:**
- 새로운 건강식품 출시
- 언제 방송하면 좋을지 모름

**해결 방법:**
1. 여러 시간대로 예측 실행
2. 가장 높은 매출 시간대 선택
3. 해당 시간에 편성

**결과:**
- 최적 시간대 선택으로 매출 극대화
- 데이터 기반 의사결정

---

### 시나리오 2: 재방송 편성 최적화

**상황:**
- 과거에 방송했던 상품 재방송
- 이전과 다른 시간대에 방송 고려

**해결 방법:**
1. 과거 방송 시간대 매출 확인
2. 다른 시간대 예측 비교
3. 더 나은 시간대 발견 시 변경

**결과:**
- 같은 상품으로 더 높은 매출 달성

---

### 시나리오 3: 경쟁 상품 비교

**상황:**
- 같은 시간대에 여러 상품 중 하나 선택

**해결 방법:**
1. 각 상품별로 예측 실행
2. 예상 매출 비교
3. 가장 높은 상품 선택

**결과:**
- 최적 상품 선택으로 매출 극대화

## 📈 예측 결과 해석 방법

### 예측 매출액
- **단위**: 원 (KRW)
- **의미**: 해당 방송에서 예상되는 총 매출
- **예시**: 23,311,279원 → 약 2,331만원

### 신뢰도 (Confidence)
- **범위**: 0% ~ 100%
- **의미**: 예측의 정확도

| 신뢰도 | 의미 | 방송 횟수 |
|--------|------|----------|
| 90% 이상 | 매우 정확 | 20회 이상 |
| 80~90% | 정확 | 6~20회 |
| 70~80% | 보통 | 1~5회 |
| 60~70% | 참고용 | 0회 (신상품) |

### 오차 범위
- **평균 오차**: ±300만원
- **최대 오차**: ±650만원

**실제 예시:**
```
예측: 2,000만원
실제: 1,700만원 ~ 2,300만원 (정상 범위)
```

## ⚠️ 주의사항

### 1. 예측이 정확하지 않을 수 있는 경우

**신상품 (방송 0회):**
- 과거 데이터 없음
- 카테고리 평균으로만 예측
- 오차 범위 큼 (±500만원)

**특수 상황:**
- 갑작스러운 트렌드 변화
- 경쟁사 특별 이벤트
- 예상치 못한 이슈

### 2. 시간대별 차이가 큰 이유

**새벽 vs 저녁 비교:**
```
새벽 3시: 20만원
저녁 9시: 700만원

→ 35배 차이! (시청률 차이)
```

**이유:**
- 시청률이 시간대별로 크게 다름
- 저녁 시간대가 가장 높음
- 새벽 시간대는 매우 낮음

### 3. 가격 0원 상품 (보험, 여행)

**의문:**
- 가격이 0원인데 매출이 나온다?

**설명:**
- 보험/여행 상품은 수수료로 수익 발생
- 시스템이 수수료를 매출로 계산
- 정상적인 예측 결과

### 자주 묻는 질문 (FAQ)

**Q1. 신상품도 예측 가능한가요?**
- A: 네, 가능합니다. 단, 과거 데이터가 없어 정확도는 낮습니다 (카테고리 평균 기반).

**Q2. 얼마나 정확한가요?**
- A: 평균적으로 실제 매출의 ±300만원 오차입니다. 방송 횟수가 많을수록 더 정확합니다.

**Q3. 어떤 정보가 필요한가요?**
- A: 상품 코드, 방송 날짜, 방송 시간만 있으면 됩니다.

**Q4. 미래 날짜도 예측 가능한가요?**
- A: 네, 가능합니다. 날짜 제한은 없습니다.

**Q5. 여러 상품을 한 번에 비교할 수 있나요?**
- A: 네, 개발팀에 요청하시면 여러 상품을 한 번에 비교해드립니다.

**Q6. 경쟁사 편성 정보도 고려하나요?**
- A: 현재 버전에서는 자사 데이터만 사용합니다. 향후 업데이트 예정입니다.

**Q7. 예측 결과를 엑셀로 받을 수 있나요?**
- A: 네, 개발팀에 요청하시면 엑셀 파일로 제공 가능합니다.

**Q8. 과거 예측 정확도를 확인할 수 있나요?**
- A: 네, 개발팀에 요청하시면 과거 예측 vs 실제 매출 비교 리포트를 제공합니다.

## 📊 예측 결과 활용 팁

### 1. 시간대 선택

**원칙:**
- 저녁 시간대 (18:00~23:00) → 가장 높은 매출
- 오전 시간대 (09:00~12:00) → 중간 매출
- 오후 시간대 (14:00~17:00) → 낮은 매출
- 새벽 시간대 (00:00~06:00) → 매우 낮은 매출

**예외:**
- 특정 카테고리는 오전이 더 좋을 수 있음 (예: 건강식품)
- 주말 vs 평일 차이 고려
- 공휴일 특수 고려

### 2. 상품 선택

**같은 시간대에 여러 상품 중 선택:**
1. 각 상품별 예측 실행
2. 예상 매출 비교
3. 신뢰도도 함께 고려
4. 최종 결정

**예시:**
```
시간: 저녁 9시

상품 A: 2,000만원 (신뢰도 90%) ⭐
상품 B: 2,100만원 (신뢰도 60%)

→ 상품 A 선택 (신뢰도가 더 높음)
```

### 3. 편성표 최적화

**전략:**
1. 하루 전체 편성표 예측
2. 예상 총 매출 확인
3. 낮은 시간대 상품 교체
4. 재예측 후 비교
5. 최적 편성표 완성

## 📚 용어 설명

### 상품 코드
- 8자리 숫자로 된 상품 고유 번호
- 예: 15750903

### 방송 테이프
- 방송에 사용되는 영상 콘텐츠
- 하나의 상품에 여러 테이프 존재 가능

### 매출총이익 (Gross Profit)
- 방송을 통해 발생한 총 수익
- 시스템이 예측하는 값

### 카테고리
- **대분류**: 의류, 건강식품, 생활가전 등
- **중분류**: 일반의류, 건강보조식품 등
- **소분류**: 원피스/드레스, 다이어트 등

### 시간대 (Time Slot)
- **새벽**: 00:00~05:59
- **오전**: 06:00~11:59
- **오후**: 12:00~17:59
- **저녁**: 18:00~23:59

### 계절
- **봄**: 3~5월
- **여름**: 6~8월
- **가을**: 9~11월
- **겨울**: 12~2월
