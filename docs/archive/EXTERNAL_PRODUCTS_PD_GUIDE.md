# 외부 상품 (네이버 베스트) - PD 활용 가이드

## 📋 개요

방송 편성 추천 시스템에 **네이버 쇼핑 베스트 상품 TOP 20**이 추가되었습니다.  
PD님들이 외부 트렌드를 참고하여 더 나은 편성 결정을 내릴 수 있도록 도와드립니다.

---

## 🎯 주요 기능

### 1. **실시간 트렌드 파악**
- 네이버 쇼핑에서 **가장 많이 구매하는 상품 TOP 20**
- 매일 새벽 2시 자동 업데이트
- 전일 대비 순위 변동 추적 (↑3, ↓2, →, 신규)

### 2. **자동 제공**
- 방송 편성 추천 API 호출 시 **자동으로 포함**
- 별도 설정 불필요

### 3. **풍부한 정보**
- ✅ 순위 및 순위 변동
- ✅ 상품명, 이미지
- ✅ 가격 (정가, 할인가, 할인율)
- ✅ 리뷰 (평점, 리뷰 수)
- ✅ 배송 정보 (무료배송 여부, 배송비)
- ✅ 판매자 정보

## 🎯 활용 방법

### 1. **트렌드 파악**

#### 급상승 상품 확인
- **순위 변동**: `↑5` 이상 → 급상승 트렌드
- **예시**: 3위 → 1위 (↑2) → 최근 인기 급상승

**활용**:
```
"가습기가 1위로 급상승했네요. 겨울철 건조한 날씨 때문인 것 같습니다.
우리도 가습기 관련 상품을 편성하면 좋을 것 같아요."
```

#### 신규 진입 상품
- **표시**: `신규` 배지
- **의미**: 어제는 TOP 20에 없었는데 오늘 새로 진입

**활용**:
```
"과메기가 신규로 4위에 진입했네요. 계절 상품이라 지금이 적기인 것 같습니다."
```

---

### 2. **가격 경쟁력 분석**

#### 할인율 확인
- **할인율 30% 이상**: 소비자 관심도 높음
- **예시**: 정가 62,900원 → 할인가 35,900원 (42% 할인)

**활용**:
```
"사과가 42% 할인으로 6위에 올라왔네요. 
우리 과일 상품도 할인 프로모션을 고려해볼까요?"
```

#### 가격대별 분포
- **저가**: 1만원 이하 (일상 소비재)
- **중가**: 1~10만원 (가전, 식품)
- **고가**: 10만원 이상 (전자제품, 명품)

**활용**:
```
"TOP 10에 중가 상품이 많네요. 
우리도 5~10만원대 상품 비중을 늘려볼까요?"
```

---

### 3. **리뷰 분석**

#### 고평가 상품
- **평점 4.8 이상 + 리뷰 1만개 이상**: 검증된 상품
- **예시**: ⭐4.9 (35,093개 리뷰)

**활용**:
```
"가습기가 4.9점에 리뷰 3만5천개네요. 
품질이 검증된 상품이니 우리도 유사 상품을 찾아볼까요?"
```

#### 신제품 vs 검증 상품
- **리뷰 적음**: 신제품 또는 틈새 상품
- **리뷰 많음**: 검증된 인기 상품

**활용**:
```
"아이폰은 리뷰가 461개로 적지만 6위네요. 
신제품이라 관심이 높은 것 같습니다."
```

---

### 4. **카테고리 트렌드**

#### 인기 카테고리 파악
- TOP 20에 많이 등장하는 카테고리 확인
- **예시**: 식품 7개, 가전 5개, 전자제품 3개...

**활용**:
```
"식품 카테고리가 7개나 들어있네요. 
김장철이라 김치, 배추 관련 상품이 인기인 것 같습니다."
```

#### 계절/시기별 변화
- **겨울**: 가습기, 난방용품, 김장 식품
- **여름**: 선풍기, 에어컨, 냉장고
- **명절**: 선물세트, 식품

**활용**:
```
"11월이라 김장 관련 상품이 많네요. 
우리도 김장 특집 편성을 고려해봅시다."
```

---

### 5. **경쟁사 분석**

#### 외부 vs 내부 상품 비교
- 네이버 베스트 상품과 우리 추천 상품 비교
- 가격, 품질, 카테고리 차이 분석

**활용**:
```
"네이버에서는 가습기가 1위인데 우리는 김치를 1순위로 추천했네요.
가습기도 추가로 편성하면 좋을 것 같습니다."
```

#### 가격 경쟁력
- 동일/유사 상품의 가격 비교
- 우리 상품이 더 비싸면 할인 고려

**활용**:
```
"네이버에서 김치가 39,900원에 팔리는데 우리는 45,000원이네요.
가격 조정이나 프로모션을 고려해야 할 것 같습니다."
```

## 📊 실전 활용 시나리오

### 시나리오 1: 오후 2시 방송 편성

**상황**: 오후 2시 방송 편성을 위해 추천 API 호출

**내부 추천 (AI)**:
1. 정성곳간 강순의 김치 포기8kg
2. 불스원 겨울안전운전 패키지
3. 김수미 포기김치7kg

**외부 베스트 (네이버)**:
1. 윤남텍 가습기 (↑3, ⭐4.9, 35k개)
2. 해남 절임배추 (→, ⭐4.7, 41k개)
3. 아이폰 17 프로 (↓1, ⭐4.9, 461개)

**PD 판단**:
```
✅ 내부 추천 1, 3번 (김치) → 외부에서도 김치/배추가 인기 → 채택
✅ 외부 1번 (가습기) → 급상승 중 → 추가 편성 고려
⚠️ 외부 3번 (아이폰) → 고가 상품, 우리 타겟과 맞지 않음 → 제외
```

**최종 편성**:
1. 정성곳간 강순의 김치 (내부 추천 + 외부 트렌드 일치)
2. 윤남텍 가습기 (외부 급상승 트렌드)
3. 김수미 포기김치 (내부 추천)

---

### 시나리오 2: 긴급 편성 변경

**상황**: 오전 10시 방송 중 시청률 저조

**외부 베스트 확인**:
- 1위: 가습기 (↑5) ← 급상승!
- 4위: 과메기 (신규) ← 새로 진입!

**PD 판단**:
```
"가습기가 급상승 중이고 과메기가 신규 진입했네요.
오후 방송에 이 두 상품을 긴급 편성해봅시다."
```

**결과**:
- 트렌드에 맞춰 빠르게 대응
- 시청률 및 매출 향상 기대

---

### 시나리오 3: 주간 편성 회의

**상황**: 다음 주 편성 계획 수립

**외부 베스트 분석 (최근 7일)**:
- **식품**: 김치, 배추, 과메기 (계속 상위권)
- **가전**: 가습기, 공기청정기 (겨울철 수요)
- **전자제품**: 아이폰, 갤럭시 (신제품 출시)

**PD 판단**:
```
"다음 주는 김장 특집과 겨울 가전 특집을 준비합시다.
아이폰 같은 고가 전자제품은 우리 타겟과 맞지 않으니 제외하고,
중저가 가전 제품 위주로 편성합시다."
```

## 🎯 주의사항

### 1. **외부 트렌드는 참고용**
- 네이버 베스트 ≠ 우리 방송에 최적
- 우리 시청자층, 상품 재고, 방송 시간대 고려 필요

### 2. **내부 추천과 균형**
- AI 추천 (내부): 매출 예측 기반
- 외부 베스트: 시장 트렌드 반영
- **두 가지를 종합하여 최종 판단**

### 3. **가격 경쟁력 확인**
- 동일 상품이 네이버에서 더 저렴하면 경쟁력 없음
- 가격 조정 또는 프로모션 필요

### 4. **재고 확인**
- 외부에서 인기 있어도 우리 재고가 없으면 편성 불가
- 긴급 발주 가능 여부 확인
