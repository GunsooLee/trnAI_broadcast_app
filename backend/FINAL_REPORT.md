| 조언 | 검증 결과 | 비고 |

|------|------------|----------|----------|---------|

- **과적합 발생**

### 4️⃣ **Train-Serve Skew 원인 파악** ✅

**핵심 피처 `product_avg_profit` 분포 비교:**

| 통계 | 학습 데이터 | 실제 추론 | 차이 |
|------|------------|----------|------|
| **평균** | 14,026,733원 | 18,270,033원 | **+30.3%** 🔴 |
| **중앙값** | 14,840,435원 | 19,422,680원 | **+30.9%** 🔴 |
| **표준편차** | 9,090,582원 | 9,764,019원 | +7.4% |

**원인:**
- 학습 데이터: 평균적인 상품들 포함
- 실제 방송: **고성과 상품만 선택**되어 방송
- 예: 참좋은여행(27,550,000원), 대원제약 알부민(32,370,350원)

**결론:**
- **선택 편향(Selection Bias)** 문제
- 실제 방송되는 상품은 이미 과거 성과가 좋은 상품들
- 모델이 학습한 분포와 실제 분포가 근본적으로 다름

## 💡 근본 원인 및 해결 방안

### 🔴 **근본 원인: 선택 편향 (Selection Bias)**

**문제:**
```
학습 데이터 = 모든 상품 (평균 성과)
실제 방송 = 고성과 상품만 (평균보다 30% 높음)
```

**왜 모델이 실패하는가:**
- 모델은 "평균적인 상품"을 학습
- 실제로는 "우수한 상품"만 방송
- 모델이 본 적 없는 분포

### ✅ **해결 방안**

#### 1. **단기 해결책 (즉시 적용 가능)**

**A. 피처 정규화**
```python
# product_avg_profit을 카테고리 평균 대비 비율로 변환
product_avg_profit_normalized = product_avg_profit / category_avg_profit
```
- 절대값 대신 상대적 성과 사용
- 선택 편향 완화

**B. 앙상블 예측**
```python
# 여러 기간 모델 조합
prediction = 0.5 * model_all_data.predict(X) + 
             0.3 * model_recent_1year.predict(X) +
             0.2 * model_recent_6months.predict(X)
```

#### 2. **중기 해결책**

**A. 추가 피처 수집**
- 프로모션 여부 (할인율, 사은품)
- 방송 순서 (첫 방송 vs 재방송)
- 경쟁사 동시간대 편성
- PD 선호도 점수

**B. 타겟 변환 재검토**
```python
# Box-Cox 변환 시도
from scipy.stats import boxcox
y_transformed, lambda_param = boxcox(y + 1)
```

#### 3. **장기 해결책**

**A. 2단계 모델**
```
1단계: 방송 선택 모델 (어떤 상품이 방송될까?)
2단계: 매출 예측 모델 (방송되면 얼마나 팔릴까?)
```

**B. 도메인 적응 (Domain Adaptation)**
- 학습 데이터와 실제 데이터의 분포 차이를 모델에 반영
- Importance Weighting 기법 사용
